## 🛠️ Panduan Developer

### ⚡ Quick Start (5 menit)

```bash
# 1️⃣ Clone & install
git clone https://github.com/username/nalar-ai.git
cd nalar-ai
npm install

# 2️⃣ Setup API key
cp .env.example .env
# ✏️ Edit .env → isi GEMINI_API_KEY=...

# 3️⃣ Jalankan!
npm run dev
# → Landing: http://localhost:5173
# → Chat:    http://localhost:5173/chat.html
```

### 📋 Semua Perintah

| Perintah | Buat Apa? |
|:---------|:----------|
| `npm run dev` | 🔥 Jalankan dev server (hot reload) |
| `npm run build` | 📦 Build production ke `dist/` |
| `npm start` | 🚀 Jalankan production server |
| `npm run lint` | ✅ Cek TypeScript error |
| `npm run clean` | 🧹 Hapus folder `dist/` |

### 🔐 Environment Variables

```env
# ✨ WAJIB
GEMINI_API_KEY=ai_xxxxxxxxxxxx    # Dari https://aistudio.google.com/apikey

# ⚙️ OPSIONAL
PORT=3000                          # Port server (default: 3000)
NODE_ENV=development               # 'development' | 'production'
```

> ⚠️ **Jangan pernah commit `.env`!** Sudah di-`.gitignore`.

### 🚢 Deploy ke Vercel (3 langkah)

```
┌──────────────────────┐
│ 1. Push ke GitHub    │──▶  git push origin main
└──────────┬───────────┘
           ▼
┌──────────────────────┐
│ 2. Import di Vercel  │──▶  vercel.com → New Project
└──────────┬───────────┘
           ▼
┌──────────────────────┐
│ 3. Set Env + Deploy  │──▶  GEMINI_API_KEY=...  ✅ Done!
└──────────────────────┘
```
