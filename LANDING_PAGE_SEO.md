Jangan cuma baca. Buktikan kamu memahami.

NalarAI tidak memberi jawaban gratis — ia membimbing kamu menemukan sendiri.

### CTA Button Implementation Notes

- Setiap CTA button harus menggunakan `<a>` tag dengan `href` yang relevan, bukan `<button>` tanpa link — agar crawlable oleh search engine.
- Tambahkan `rel="noopener"` untuk external links.
- Gunakan `aria-label` yang deskriptif untuk accessibility: e.g., `aria-label="Mulai belajar gratis dengan NalarAI"`.
- Primary CTA harus visible tanpa scroll (above the fold).

## ✅ SEO Checklist

- [x] Title tag ≤ 60 karakter, mengandung keyword utama
- [x] Meta description ≤ 160 karakter, mengandung keyword + CTA
- [x] Open Graph tags lengkap (title, description, image, url, type, locale)
- [x] Twitter Card tags lengkap
- [x] Canonical URL ditentukan
- [x] Robots meta tag (index, follow)
- [x] Hreflang tags untuk multi-language support
- [x] Geo meta tags untuk targeting Indonesia
- [x] JSON-LD structured data (SoftwareApplication, WebPage, FAQPage)
- [x] H1 unik, mengandung keyword utama
- [x] Heading hierarchy terstruktur (H1 → H2 → H3)
- [x] Keyword placement di title, H1, H2, meta description, body, FAQ
- [x] Image alt text deskriptif untuk setiap visual
- [x] Internal linking dengan anchor text SEO-friendly
- [x] CTA buttons menggunakan `<a>` tag (crawlable)
- [x] FAQ section dengan schema markup (rich snippet potential)
- [x] Mobile-friendly meta tags (viewport, PWA, theme-color)
- [x] Favicon dan apple-touch-icon
- [x] Language dan content-language meta tags
